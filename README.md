# 풋살장 예약 및 랜덤매칭 사이트(Java)
 - 국비 교육간 팀프로젝트
 - Spring Framework 기반 웹페이지
 - 담당업무 : 클럽페이지(생성,가입,탈퇴,리스트) / 팀페이지(페이징) / 랜덤 매칭 / 관리자페이지(로그인, 게시글관리, 회원관리)
 - 언어 : JAVA, JSTL, EL, Spring Framework, HTML/CSS, JavaScript, J-Query, BootStrap, Mysql
 - 개발도구 : Eclipse, STS
